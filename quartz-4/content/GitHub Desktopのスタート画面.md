## 🖥️ 画面の全体：GitHub Desktopでリポジトリを始めよう！

**英語原文（左上の見出し）：**

> Let's get started!  
> Add a repository to GitHub Desktop to start collaborating

**和訳：**

> はじめましょ！  
> GitHub Desktopにリポジトリ（プロジェクト）を追加して、共同作業を始めましょう！

---

## 📂 中央の選択肢一覧

### 🔷 **Create a tutorial repository...**

→ **チュートリアル用のリポジトリを作る**（Gitの使い方を練習できる）

---

### 🧲 **Clone a repository from the Internet...**

→ **インターネット上のリポジトリを複製（クローン）する**  
👉 GitHub上の誰かのプロジェクトを自分のPCにコピーする機能！

---

### 🆕 **Create a New Repository on your local drive...**

→ **自分のPC上に新しいリポジトリを作る**  
👉 ローカルで新しいGitプロジェクトを始めたいとき！

---

### 📁 **Add an Existing Repository from your local drive...**

→ **すでにPCにあるリポジトリをGitHub Desktopに追加する**  
👉 もう作ってあるフォルダをGUI管理したいとき！

---

## 🧠 ProTip（プロチップ）

> → **すでにあるGitフォルダをこのウィンドウにドラッグ＆ドロップすれば、GitHub Desktopに追加できるよ！**

---

## 💡 つまりこの画面でできること：

|やりたいこと|どのボタンを押せばいい？|
|---|---|
|GitHub上のプロジェクトをコピーしたい|Clone from the Internet|
|自分のPCで新規プロジェクトを始めたい|Create a New Repository|
|すでにあるフォルダをGitで管理したい|Add Existing Repository|
|チュートリアルで練習したい|Create a tutorial repository|
