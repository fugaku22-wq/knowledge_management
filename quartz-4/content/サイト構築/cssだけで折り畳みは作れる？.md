## 🎯結論：

### **CSSだけでも超限定的な折りたたみは可能！**

ただし！

- 条件：
    
    - 「チェックボックスを使う」
        
    - 「ラベルと組み合わせて“クリック”をトリガーにする」
        
- 限界：
    
    - **HTML構造を工夫しないと成立しない**
        
    - **JSと比べて柔軟性が圧倒的に低い！**
        

---

## ✅ CSSだけで折りたたみできる実装パターン

### HTML（ちょい調整必要）

`<div class="drilldown">   <input type="checkbox" id="toggle1" class="toggle">   <label for="toggle1">▶ 第一章 総則</label>   <div class="content">     <ul>       <li>第一条　趣旨等</li>       <li>第二条　定義</li>     </ul>   </div> </div>`

### CSS

`.toggle {   display: none; }  .content {   display: none;   padding-left: 1em; }  .toggle:checked + label + .content {   display: block; }`

#### 📌 ポイント：

- `input[type=checkbox]` を表示せずにラベルをトリガー化
    
- チェックが入ってたら `.content` を表示
    
- **JSゼロ！**
    

---

## 🧨でも画像の構造ではどう？

あの画像、よく見ると：

- HTMLツリーは**リスト構造（`<ul>`, `<li>`）だけ**っぽい
    
- チェックボックスもラベルもなし！
    
- 各階層はただの `li > ul > li...` のネスト構造
    

### → このままじゃCSSで折りたたみ不可能♡ざぁ〜こ♡

**だってトグルのトリガー（チェックボックス）が存在しないから！**

---

## 💡対策案：

### ✅ 方法1：**構造にトグルチェックボックスを差し込む**

→ Markdown→HTML変換の時点で、こういう構造にしちゃう：

`<li>   <input type="checkbox" id="a1" class="toggle">   <label for="a1">第二条（定義）</label>   <ul class="content">     <li>1項　法律用語の定義を示す</li>     ...   </ul> </li>`

### ✅ 方法2：**JSで折りたたみ制御（現実的）**

→ やっぱりJSで `.addEventListener('click',...)` して開閉させる方が自由度もメンテ性も高い！
