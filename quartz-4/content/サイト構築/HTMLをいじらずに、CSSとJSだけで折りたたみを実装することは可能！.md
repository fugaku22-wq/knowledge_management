ただし…  
**“構造に規則性がある”場合に限る♡**  
つまり、**「どの要素を折りたたみたいか」をCSSやJSが特定できる状態**じゃないとムリ〜♡

---

## 🎯どういう仕組み？

### ✅ HTMLの構造はそのまま

たとえばこういうネストリストがあるとして：

`<ul>   <li>     第一章 総則     <ul>       <li>第一条（趣旨等）</li>       <li>第二条（定義）         <ul>           <li>1項　法律用語の定義</li>         </ul>       </li>     </ul>   </li> </ul>`

このとき「親の`li`をクリックしたら、中の`ul`が出入りする」って動作を  
**JSで仕掛けて、CSSで隠す／見せる**ようにするの♡

---

## 🛠やり方イメージ（HTML無改造！）

### 💡CSS（最初は全部折りたたむ）

`li > ul {   display: none;   margin-left: 1em; } li.open > ul {   display: block; } li {   cursor: pointer; }`

### 💡JS（クリックされたら開閉）

`document.querySelectorAll("li").forEach(li => {   li.addEventListener("click", e => {     // 子ulがあるliだけ対象     if (li.querySelector("ul")) {       li.classList.toggle("open");       e.stopPropagation(); // ネストでバブルしないように     }   }); });`

### ✅これで実現できる！

- `li`要素がクリックされたら
    
- 中の`ul`を表示・非表示に切り替え
    
- `li.open`にスタイルがついて展開される♡
    

---

## 📌注意点！

|限界|内容|
|---|---|
|🔹 タイトルと中身が混ざってる|タイトル部分も中身と一緒にクリック対象になる|
|🔹 細かいトグル制御できない|ラベル・ボタン・セクション分割がないと制御しづらい|
|🔹 装飾も制限あり|アイコン変化・アニメーション等は追加CSSが必要になる|

---

## 🧠まとめ

|目的|HTML変更なしでできる？|方法|
|---|---|---|
|折りたたみの実装|✅ できる|JSでクラス操作＋CSSで表示制御|
|完全なUI制御（ボタン分離等）|❌ 工夫が必要|HTML側に微修正 or クラス付与必須|
