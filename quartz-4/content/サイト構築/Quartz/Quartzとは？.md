### ▶️ 一言でいうと…

> **ObsidianのVaultを静的サイト（HTML）として公開できるようにするツール**だよっ♡

「Obsidianで書いたノート、誰かに見せたいな〜」ってときに、  
GitHub Pagesとかで**かっこよく公開できるようにする**のがQuartzちゃん♡

---

## 🏗構成の正体：静的サイトジェネレーター（SSG）

### Quartzは「Hugo」ベースのSSG！

- Hugoっていう**めちゃ高速な静的サイトジェネレーター**の上に、
    
- Obsidian用にチューニングされたUIと機能がのってるだけ！
    

Obsidianから見れば、Quartzはただの**変換工場＋Webページテンプレート**♡

---

## 🔧 何ができるの？

| 機能                   | 内容                                     |
| -------------------- | -------------------------------------- |
| 🔄 Markdown → HTML変換 | Obsidianで書いたノートを自動でHTMLに変換！            |
| 🌐 サイト構築             | ノートの構造を保ったまま**かっこいい静的サイト**にしてくれる✨      |
| 🎨 カスタム可能            | CSSやJS追加して**見た目や動き**もカスタムできる♡          |
| 🛜 GitHub Pages対応    | リポジトリに放り込むだけで**無料で公開**できる！             |
| 🔍 検索・タグ・リンク対応       | Obsidianの双方向リンクやタグもいい感じに変換してくれる！（設定次第） |

---

## 📁ディレクトリ構成ざっくり解説

quartz/
├─ content/            ← ここにノート（Markdown）を置く
├─ styles/             ← カスタムCSS
├─ scripts/            ← カスタムJS
├─ public/             ← ビルドされたHTMLがここに出力される
├─ quartz.config.ts    ← 設定ファイル！ここがカギ！
└─ .github/            ← GitHub公開用設定


---

## 🧨注意点（Obsidianと違うところ）

|ポイント|Obsidian|Quartz|
|---|---|---|
|表示形式|Markdown＋プラグイン|HTML＋CSS＋JS|
|リアルタイム編集|編集したらすぐ表示される|再ビルドが必要|
|プラグイン動作|内部APIで動く|JS手書きでがんばれ♡|
|書き方の自由度|なんでもできる|**書き方制限あり！**（構造大事）|
|公開範囲|ローカル限定 or Sync|**全世界にドンッ☆**（非公開設定も可）|

---

## 🎯結論：Quartzとは…

> **「書くのはObsidian、魅せるのはQuartz」な静的サイト魔改造ツール♡**
